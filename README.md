1.1 Назначение системы
Разработка Python-приложения для автоматического парсинга веб-страниц, обнаружения рекламных слотов и создания скриншотов с визуальным выделением рекламных областей.

1.2. Цели проекта
Создание инструмента для автоматического анализа веб-страниц на наличие рекламных блоков\слотов
Разработка системы визуализации обнаруженной рекламы через аннотированные скриншоты
Обеспечение возможности анализа рекламных сетей и технологий
Создание отчетной системы для сбора статистики

2. Общее описание системы

2.1. Функциональность системы
Система должна предоставлять следующие возможности:
Пакетная обработка URL
Загрузка и анализ веб-страниц по URL
Обнаружение рекламных слотов различных типов
Идентификация рекламных сетей и технологий
Создание скриншотов страниц и отдельных рекламных слотов
Визуальное выделение рекламных областей на скриншотах
Генерация отчетов после окончания работы системы


3. Технические требования

3.1.1. Модуль парсинга страниц
Загрузка веб-страниц с поддержкой JavaScript
Извлечение и анализ HTML-контента
Обнаружение рекламных элементов по различным признакам(размеры, HTML шаблон контента, ссылки, скрипты)
Анализ DOM-структуры страницы
Обработка iframe и динамического контента

3.1.2. Модуль обнаружения рекламы
Распознавание рекламных слотов по HTML-атрибутам
Обнаружение рекламных скриптов и iframe
Идентификация рекламных сетей (Google, Yandex и др.)
Анализ размеров и позиционирования элементов
Определение типа рекламы (баннер, видео, нативная и т.д.)

3.1.3. Модуль создания скриншотов
Захват полной страницы или видимой области
Аннотирование с выделением рекламных зон
Добавление подписей и легенд

3.1.4. Модуль эмуляции взаимодействия с рекламой
Анализ кликабельности рекламного слота
Выбор стратегии взаимодействия
Анализ перехода
Создание скриншотов перехода
Анализ кликовой ссылки перехода и выделение GET параметров UTM

3.1.5. Модуль отчетности
Генерация структурированных отчетов
Экспорт в JSON, CSV, PDF форматы



3.2. Нефункциональные требования
Потребление памяти: не более 1 ГБ на процесс
Устойчивость к таймаутам и ошибкам загрузки
Обработка ошибок загрузки страниц
Устойчивость к динамическим изменениям структуры сайтов
Логирование операций и ошибок
Валидация входных URL
